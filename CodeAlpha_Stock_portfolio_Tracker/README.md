# 📈 Stock Portfolio Tracker

## 📖 Overview

The Stock Portfolio Tracker is a Python-based application that helps users calculate the total value of their stock investments. Users can enter stock symbols and quantities, and the program calculates the investment value based on predefined stock prices.

## ✨ Features

- Accepts multiple stock entries from the user
- Uses a dictionary to store stock prices
- Calculates individual stock investment values
- Calculates total portfolio value
- Handles invalid stock symbols
- Saves portfolio details to a text file

## 🛠 Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements
- File Handling

## ⚙️ How It Works

1. Enter the number of stocks.
2. Enter the stock symbol and quantity for each stock.
3. The program validates the stock symbol.
4. Investment values are calculated automatically.
5. A portfolio summary is displayed.
6. The summary is saved to `portfolio_summary.txt`.

## 📊 Sample Stocks

- AAPL
- TSLA
- GOOGL
- MSFT
- AMZN

## 📋 Output

The program generates a portfolio summary showing:

- Stock Symbol
- Quantity
- Investment Value
- Total Investment Value

The summary is also stored in a text file named `portfolio_summary.txt`.
