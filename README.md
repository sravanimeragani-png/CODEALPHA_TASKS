# 🎯 Hangman Game

A simple text-based **Hangman Game** developed in Python as part of the **CodeAlpha Internship Task 1**. In this game, the player tries to guess a hidden word one letter at a time before running out of attempts.

## 📌 Objective

The goal of the game is to correctly guess the secret word by entering one letter at a time. The player has a maximum of **6 incorrect guesses** to find the word.

## ✨ Features

- Randomly selects a word from a predefined list
- Allows letter-by-letter guessing
- Tracks correct and incorrect guesses
- Displays the current progress of the word
- Limits the player to 6 wrong attempts
- Simple and beginner-friendly console interface

## 🛠️ Technologies Used

- Python
- Random Module

## 📚 Concepts Covered

- Variables and Data Types
- Lists
- Strings

- Loops (`while`)
- User Input Handling
- Random Word Selection

## 🎮 How to Play

1. Run the Python program.
2. A hidden word will be selected randomly.
3. Enter one letter at a time.
4. If the letter is correct, it will be revealed in the word.
5. If the letter is incorrect, one attempt will be deducted.
6. Guess the entire word before using all 6 attempts.
