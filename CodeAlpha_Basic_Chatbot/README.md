# 🤖 Basic Chatbot

## 📖 Description

A simple rule-based chatbot developed using Python. The chatbot interacts with users by responding to predefined commands and questions. This project demonstrates the use of basic Python programming concepts such as functions, loops, conditional statements, and user input/output.

## ✨ Features

- Responds to greetings and basic queries
- Displays current date and time
- Provides a help menu
- Handles invalid inputs
- Exits the chat when the user enters `"bye"`

## 🛠 Technologies Used

- Python 3

## 📚 Concepts Used

- Functions
- if-elif-else Statements
- While Loops
- User Input and Output
- datetime Module

## 🎯 Outcome

This project successfully implements a basic chatbot capable of interacting with users through predefined responses and demonstrates fundamental Python programming concepts.
